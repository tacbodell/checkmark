## About The Project

**CheckMark** is a full-stack task tracking application built to be free, easy-to-use, and accessible to those with disabilities. Many of the alternatives on the market either require a subscription, lack crucial functionalities like recurring tasks, or are inaccessible to those with vision difficulties. CheckMark solves this problem by providing access to organized tasks throughout the day, whether it be business- or home- related. CheckMark is free, open-source, and forkable to be bent and adapted to anybody's needs.
